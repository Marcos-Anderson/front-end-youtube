# Clone da Interface do YouTube 🎥

Este projeto é uma recriação da interface principal do YouTube desenvolvida para praticar conceitos avançados de **HTML5** e **CSS3**,
como CSS Grid, Flexbox e manipulação de SVGs. O objetivo foi replicar fielmente o layout moderno da plataforma, focando em organização de código e responsividade.

## 🚀 Tecnologias Utilizadas
- **HTML5**: Estruturação semântica para melhor acessibilidade.
- **CSS3**: Utilização de **CSS Grid** e **Flexbox** para o layout dinâmico das miniaturas e menus.
- **SVG**: Ícones vetoriais integrados diretamente no código para garantir nitidez e performance.

## 🛠️ Funcionalidades Reproduzidas
- **Cabeçalho Fixo**: Barra de pesquisa estilizada com design oval e ícones funcionais.
- **Menu Lateral**: Organização por categorias com estados de seleção (hover e active).
- **Grid de Vídeos**: Layout responsivo que organiza títulos, canais e metadados.
- **Componentes Customizados**: Estilização de selos "Ao Vivo" e botões de filtro.
